# Project Rules & Preferences

## Slide Design
- **Agenda**: Background must be WHITE (`#FFFFFF`).
- **Footer**: Do NOT include company name (Keep it empty).
- **Placeholders**: If no content is provided for a placeholder (Title, etc.), remove the placeholder element entirely instead of setting empty text.

## Workflows
- **Deployment**: Run `npm run build` and `clasp push`. (Automated via `/deploy`)
