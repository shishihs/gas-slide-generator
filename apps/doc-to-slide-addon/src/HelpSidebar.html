ユーザーガイド - スライド起草くん

このアドオンは、Google ドキュメントの原稿から、AI を活用してその場で Google Slides を自動生成するツールです。

✨ 主な機能

  • テンプレート自動生成: 最初に一度だけ、デザインのベースとなるスライドを作成します。
  • AI 解析: ドキュメントの文章を読み取り、スライド構成案 (JSON) を自動で作ります。
  • スライド生成: 構成案を元に、実際の Google Slides を生成します。

🚀 はじめ方 (Quick Start)

Step 1: テンプレートの準備

まず、スライドのデザインひな形（テンプレート）を準備します。

  • メニューから スライド起草くん > 設定 を開きます。
  • 「スライドテンプレート」セクションにある 「テンプレートを新規作成」ボタン をクリック。
  • 「スライド作成完了」と表示されたら、作成された URL を開きます。
  • 生成されたスライドの「テーマ」を変更して、お好みのデザイン（フォントや色）に調整してください。

- ※このスライドが今後の生成のベースになります。

Step 2: JSON の準備 (手動モード - 推奨)

推奨されるデフォルトの使い方です。専用の Gem (チャットボット) を使用します。

  • Google Gemini (または設定画面に保存した Gem URL) を開きます。
  • 原稿を渡し、「スライドジェネレーター用のJSONを作って」と依頼します。
  • 生成された JSON コードをコピーします。

Step 3: ドキュメントに貼り付け

  • この Google ドキュメントに、コピーした JSON をそのまま貼り付けます。

- ※ 元の原稿は残していても構いませんが、JSON 全体が正しく貼られていることを確認してください。

Step 4: スライド生成

  • メニューから スライド起草くん > スライド生成 をクリック。
  • ドキュメント内の JSON が読み込まれ、スライドが自動生成されます。

💡 自動生成モード (Advanced)

GCP プロジェクトをお持ちの方は、Vertex AI と連携してドキュメントから直接スライド生成が可能です。

  • メニューから スライド起草くん > 設定 を開き、GCP Project ID などを入力します。
  • ドキュメントに原稿 (文章) を入力します。
  • スライド生成 メニューを実行すると、AI が自動で JSON 変換〜スライド生成を行います。

⚙️ 設定について (Settings)

スライド起草くん > 設定 から各種設定を行えます。
  • GCP Project ID: AI 機能 (Vertex AI) を利用するための Google Cloud プロジェクト ID。
  • Region: AI のデータセンター場所 (例: asia-northeast1)。
  • Model: 使用する AI モデル (例: gemini-2.5-flash)。
  • Gem URL: 手動モードで使用する Gem のリンク。
  • テンプレートスライド: 生成のベースとなるスライドの URL または ID。

💡 ヒント: 「手動モード」で利用する場合は、GCP 設定は不要です。Gem 等のチャットボットで作った JSON を直接ドキュメントに貼り付けて利用します。

🎨 レイアウトの詳細 (Layouts)

テンプレートスライドには、以下の5つのレイアウト名のタグ (LAYOUT:NAME) が埋め込まれている必要があります。
※ スライド起草くん > 設定 > テンプレートを新規作成 でこれらが含まれたテンプレートが自動作成されます。

  • TITLE: 表紙
  • AGENDA: 目次
  • SECTION: セクション区切り
  • CONCLUSION: まとめ
  • CONTENT: 上記以外のコンテンツ (基本)

JSON type との対応

JSONを手動で編集する場合、layout プロパティは以下のように指定します。

  • title タイプ → TITLE レイアウト
  • agenda タイプ → AGENDA レイアウト
  • section タイプ → SECTION レイアウト
  • conclusion タイプ → CONCLUSION レイアウト
  • その他すべて (content, cards, timeline, kpi 等) → CONTENT レイアウト

❓ よくある質問

Q. エラーが表示されます

A. 設定画面で GCP Project ID が正しく入力されているか確認してください。また、初めて使う場合は初期設定 (テンプレート作成) が必要です。

Q. デザインを変えたい

A. 設定画面からテンプレートスライドを開き、テーマやフォントを変更してください。生成されるスライドは常にテンプレートのデザインを引き継ぎます。
